# Novelties
- In this paper, an Adaptive Model Predictive Control with Extended State Observer (AMPCESO) framework is established.
  - Evidence Section: `Method/Discussion context`
- Due to the diﬀerences in the bandwidth of velocity and attitude loops, it is diﬃcult to use a single one hidden layer network to identify two diﬀerent time scale dy- namics, therefore, this paper proposes dual single hidden layer neural networks in the predictive model in the model predictive control framework.
  - Evidence Section: `Method/Discussion context`
- The stability of the proposed AM- PCESO is rigorously proved using the Lyapunov theorem.
