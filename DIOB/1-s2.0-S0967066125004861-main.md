In this paper the robust trajectory tracking control problem for a quadrotor aircraft is addressed by utilizing the 
estimates from an Enhanced Extended State Observer (EESO). More speciпђБcally, пђБrst, the quadrotor dynamics

and input vectors, respectively, an EESO is derived allowing an asymptotic estimation of the state vector even 
when the rate of change of the unknown disturbances is fast and not negligible. Finally, on the basis of the EESO 
estimates, a robust trajectory tracking controller is designed, ensuring the asymptotic convergence of system 
state trajectories to the desired ones. The full asymptotic stability is demonstrated analytically. Simulation results 
on a quadrotor model through a high-пђБdelity low-altitude simulation environment are shown to illustrate the 
eпђГciency of the proposed method also with respect to existing solution (Extended State Observer based Control).

If the system state is not fully accessible, methods that simulta-

Among these, the scientiпђБc community has paid particular attention in

composed by an Extended State Observer, which has the task of estimat-

systems with rapidly пђВuctuating uncertainties, implying less eпђГcient es-

implies a BIBO stability of the corresponding closed-loop system.
Within this context, starting from the results obtained in Pedone and

and not negligible, which in turn implies the construction of an EESO-
based robust control law that ensures the asymptotic convergence of 
system state trajectories to the desired ones. To validate the proposed 
solution and its real-time implementability, a series of simulations on a 
low-altitude scenario are conducted in Matlab&Simulink environment 
through the Aerospace Toolbox (VDB et al., n.d.), using a low-cost hard-
ware setup consisting of a Raspberry PI 4 Model B system (Raspberry et 
al., n.d.). The simulation results highlighted the superiority of the pro-

eso-based robust control.

as follows: 1) Starting from the knowledge of the output and input sig-

a quadrotor aircraft allowing an asymptotic estimation of system state 
and of the unknown input disturbances; 2) A robust control that exploits 
the EESO estimates is designed to overcome the BIBO stability issue of 
the classical ESO-based approach; 3) Analytical proof of closed-loop sta-
bility is provided; 4) Simulations to support the theoretical results are 
carried out on a nonlinear quadrotor model via the Aerospace toolbox in

7.  Conclusion
In this paper a robust control of quadrotor trajectories in a low al-
titude environment is addressed by using an Enhanced Extended State 
Observer. Using the output and known input vectors, respectively, the 
asymptotic unknown input reconstruction is derived through an alge-
braic approach and allows to relax the existence condition of the clas-
sical extended state observer. It implies better estimation performance 
and, consequently, better control system performance especially when 
the degree of robustness required is particularly high. The full closed-
loop stability of the proposed method has been demonstrated with con-
vergence speed guarantees. Future work will be related to the construc-
tion of cooperative control systems for multiple quadrotors.
