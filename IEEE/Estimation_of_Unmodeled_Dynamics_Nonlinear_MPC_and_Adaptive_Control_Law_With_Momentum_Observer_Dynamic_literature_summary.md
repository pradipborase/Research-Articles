# Novelties
- ABSTRACT This article proposes an enhancement to estimate unmodeled dynamics within the simplified dynamic model of a quadcopter by integrating three key methodologies: Nonlinear Model Predictive Control (NMPC), a Momentum Observer Dynamics (MOD), and an adaptive control law.
  - Evidence Section: `Method/Discussion context`
- Termed as Adaptive NMPC with MOD, this integrated approach leverages NMPC, implemented using the CasADi framework, for real- time decision-making, while the momentum observer facilitates system state estimation and uncertainty mitigation.
  - Evidence Section: `Method/Discussion context`
- Through digital twin and Model in Loop (MiL) simulations, the effectiveness of this framework is demonstrated.
